# Input Text Files

This folder contains text files that can be loaded into the NER+NEL Demo application.

## Usage

1. Place your `.txt` files in this folder
2. Open the GUI application
3. Click "Load from File" button
4. Select a text file to load into the application

## Sample Files

- `sample_text.txt` - Example Serbian text with named entities

You can add your own text files here for processing with the NER+NEL models.
